# Pong-Game-in-C

Ncurses Library has been used to provide the user interface </br>
</br>
</br>
To compile the code in Terminal</br>
`gcc MyPONG.c -lncurses -o MyPONG`</br>
To run the code use</br>
`./MyPONG`</br>
